## About
OpenHound is a standardized framework for building OpenGraph collectors and converters. Built on [DLT](https://dlthub.com/docs/intro) (Data Load Tool), it provides a consistent workflow for collecting, processing, and converting data from any source into BloodHound-compatible graphs. The OpenHound-faker extension generates fake computers/users OpenGraph assets for testing BloodHound OpenGraph ingestion.
